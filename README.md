# Hibernia Research Compounds Website

Official repository for the Hibernia Research Compounds (HRC) website - a leading supplier of peptide reference materials and research compounds for laboratory and analytical research applications.

## Project Structure

```
├── index.html                 # Homepage
├── css/
│   ├── style.css             # Main stylesheet
│   └── responsive.css        # Mobile responsive styles
├── js/
│   ├── main.js              # Main JavaScript functionality
│   ├── forms.js             # Form handling and validation
│   └── navigation.js        # Navigation interactions
├── pages/
│   ├── products.html        # Product catalog
│   ├── kits.html            # KITS section
│   ├── apply-access.html    # Apply for Research Access
│   ├── lab-distributor.html # Lab & Distributor Access
│   ├── product-docs.html    # Product Documentation
│   ├── support-hub.html     # HRC Research Support Hub
│   ├── contact.html         # Contact page
│   └── terms.html           # Terms & Conditions
├── assets/
│   ├── images/
│   │   ├── logo.svg
│   │   ├── hero-bg.jpg
│   │   └── icons/

```

## Features

- ✅ Responsive design (mobile, tablet, desktop)
- ✅ Product catalog with filtering
- ✅ Research access application form
- ✅ Lab & Distributor access portal
- ✅ Support hub with FAQ
- ✅ Contact form with validation
- ✅ Terms & Conditions
- ✅ Optimized for research professionals

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. For local development, use a simple HTTP server:
   ```bash
   python -m http.server 8000
   ```
4. Navigate to `http://localhost:8000`
